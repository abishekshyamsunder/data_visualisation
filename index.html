Analysis of Ford GoBike System Data¶

by Abishek Shyamsunder¶

Investigation Overview¶

This presentation aims to provide insights into the dataset described below visually. It aims to answer the questions.

  1. What is the distribution of rides across the day?
  2. Are there any differences between the rides of Customers and Subscribers?
  3. Are there any differences between the rides across the week/ between weekdays and weekends?

Dataset Overview¶

The dataset used here is provided by Bay Area Motivate, LLC ("Motivate"), and it contains bike ride data collected between June 2017 and May 2018
The dataset is contains 1,142,896 entries and 20 columns
